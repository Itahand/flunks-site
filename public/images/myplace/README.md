# MyPlace Profile Pictures

This directory contains profile pictures for MySpace-style character profiles.

## Folder Structure:
- `the-preps/` - Profile pictures for the Prep clique
- `the-freaks/` - Profile pictures for the Freak clique  
- `the-jocks/` - Profile pictures for the Jock clique
- `the-nerds/` - Profile pictures for the Nerd clique

## File Naming:
- Main character profile: `profile.png`
- Friend pictures can be added as needed

## Upload Instructions:
1. Save your profile picture as `profile.png` in the appropriate clique folder
2. Recommended image size: 200x200px minimum for best quality
3. The geek profile picture will be automatically resized smaller (80x80px) in the interface
