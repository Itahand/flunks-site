Background Images for Semester Zero Locations

Structure
- locations/{location-key}/cover.webp (or cover.jpg as fallback)
- locations/{location-key}/{room-slug}.webp (or .jpg) for each of the 4 rooms

Location keys
arcade, jocks-house, freaks-house, geeks-house, preps-house, flunk-fm, police-station,
football-field, snack-shack, four-thieves-bar, junkyard, lake-tree, rug-doctor, shed,
secret-treehouse, high-school, paradise-motel

Room slug rules
- Lowercase, spaces and punctuation -> dash
- Examples:
  - "Main Floor" -> main-floor
  - "Prize Counter" -> prize-counter
  - "Back Room" -> back-room
  - "Chief's Office" -> chief-s-office

Recommended
- Prefer .webp for quality/size. Add .jpg fallback if needed.
- 1920x1080 or larger. Images are center-cropped to cover the screen.
